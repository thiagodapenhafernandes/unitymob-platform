Uma base visual para produtos imobiliários densos, claros e escaláveis.

Este guia traduz o layout de public/site/index.html em tokens, componentes e padrões prontos para a aplicação Rails. A intenção é manter a experiência premium da landing sem sacrificar leitura, produtividade e consistência operacional.
