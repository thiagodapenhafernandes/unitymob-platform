Unitymob Design System

Design system do Unitymob — CRM imobiliário multi-tenant, white-label (a primária é injetada por tenant via --admin-primary). Duas superfícies: Admin CRM (/admin, o foco) e Field PWA (/field). Fonte de UI Inter (+ Outfit display), Bootstrap Icons, cinzas neutros frios, primária única #365f8f.

Tudo é derivado do código real (admin_tailwind.css, camada .ax-*), não inventado. Idioma: pt-BR.
