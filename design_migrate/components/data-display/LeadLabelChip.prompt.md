**LeadLabelChip** — the color-coded tag agents attach to leads and WhatsApp conversations (e.g. "Quente", "Financiamento").

```jsx
<LeadLabelChip color="red">Quente</LeadLabelChip>
<LeadLabelChip color="cyan">Financiamento</LeadLabelChip>
```

Colors: `gray · green · amber · red · blue · purple · cyan`.
