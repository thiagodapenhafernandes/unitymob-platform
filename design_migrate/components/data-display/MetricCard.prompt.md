**MetricCard** — the KPI tile that tiles the dashboard header. Big tabular value, uppercase label, optional badge, hint line and progress bar.

```jsx
<MetricCard label="Leads hoje" value="128" badge={<Badge tone="gray">+14 em 7d</Badge>} hint="9 represados · 22 novos" />
<MetricCard label="Regras ativas" value="8/12" progress={66} />
```
