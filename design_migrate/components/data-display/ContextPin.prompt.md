**ContextPin** — the persistent entity chip in the navbar context bar. Each CRM object type has a fixed accent on the left border.

```jsx
<ContextPin type="property">Apto 302 · COD-84213</ContextPin>
<ContextPin type="lead" onRemove={() => {}}>Marina Costa</ContextPin>
```

Types: `property · owner · lead · proposal · whatsapp`.
