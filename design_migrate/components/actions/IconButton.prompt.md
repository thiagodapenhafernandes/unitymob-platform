**IconButton** — square icon-only control for dense toolbars, table-row actions and card headers. Use `bare` inside toolbars (transparent, hover fill) and `outlined` when it stands alone.

```jsx
<IconButton icon="three-dots-vertical" label="Mais ações" />
<IconButton icon="pencil" variant="outlined" label="Editar" />
```

Always pass `label` for accessibility. Icons are Bootstrap Icons names without `bi-`.
